# memories app

간단하게 카드 형식으로 기록을 남길 수 있는 앱

사용한 기술 스택 <br />
- Client-Side Stack : React, Redux, Redux-Thunk, Axios <br />
- Server-Side Stack : Node, Express, mongoose, mongoDB
